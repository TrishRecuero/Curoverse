Why People Choose Curoverse

Researchers

Using Curoverse accelerates science by freeing bioinformaticians to focus on generating reliable results instead of wrestling with computing infrastructure.

IT Leadership

With Curoverse, IT can deliver a reliable, secure, elastic computing environment that meets user needs, integrates with existing systems, and lowers costs.

Clinical Labs

The Curoverse system helps labs develop, validate, deploy, and manage pipelines for NGS-based diagnostic testing that is consistent with CLIA/CAP standards.

100% Open Source Software

Curoverse is built with Arvados, an open source system first developed at Harvard Medical School and now maintained by a community of developers and informaticians dedicated to creating open infrastructure for genomic and science and personalized medicine.
